# Skills Repo Initialization Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Initialize the skills repo as a monorepo with workspaces, compatible with `npx skills add klaushougesen/skills` and Cloudflare's .well-known agent-skills discovery protocol, with one real skill migrated.

**Architecture:** Skills live in `skills/<name>/SKILL.md` (native discovery by `npx skills`). Root `package.json` uses npm workspaces. A build script generates `.well-known/agent-skills/index.json` with SHA-256 digests per the Cloudflare RFC v0.2.0. GitHub Pages serves the `.well-known` directory so agents can discover skills via URL. Existing skills from `~/.claude/skills/` are migrated in.

**Tech Stack:** Node.js, npm workspaces, Shell scripts, agentskills.io spec v0.2.0

### Task 2: scripts/build-index.sh

**Files:**
- Create: `scripts/build-index.sh`

This script scans `skills/*/SKILL.md`, computes SHA-256 digests, and writes `.well-known/agent-skills/index.json` per the Cloudflare RFC v0.2.0 spec.

- [ ] **Step 1: Create the build script**

```bash
#!/usr/bin/env bash
set -euo pipefail

INDEX_DIR=".well-known/agent-skills"
INDEX_FILE="$INDEX_DIR/index.json"
SCHEMA="https://schemas.agentskills.io/discovery/0.2.0/schema.json"

mkdir -p "$INDEX_DIR"

# Collect skills
entries=()
for skill_dir in skills/*/; do
  [ -f "$skill_dir/SKILL.md" ] || continue

  skill_name=$(basename "$skill_dir")
  digest="sha256:$(shasum -a 256 "$skill_dir/SKILL.md" | cut -d' ' -f1)"
  description=$(awk '/^---/{c++; next} c==1 && /^description:/{sub(/^description: */, ""); print; exit}' "$skill_dir/SKILL.md")

  entries+=("    {\"name\":\"${skill_name}\",\"type\":\"skill-md\",\"description\":\"${description}\",\"url\":\"/.well-known/agent-skills/${skill_name}/SKILL.md\",\"digest\":\"${digest}\"}")
done

# Write index.json
exec 3>"$INDEX_FILE"
echo "{" >&3
echo "  \"\$schema\": \"$SCHEMA\"," >&3
echo "  \"skills\": [" >&3

for i in "${!entries[@]}"; do
  if [ "$i" -gt 0 ]; then
    echo "," >&3
  fi
  printf '%s' "${entries[$i]}" >&3
done

echo "" >&3
echo "  ]" >&3
echo "}" >&3
exec 3>&-

echo "Built $INDEX_FILE with ${#entries[@]} skills"
```

- [ ] **Step 2: Make it executable**

Run: `chmod +x scripts/build-index.sh`

- [ ] **Step 3: Verify it runs (will produce empty skills array since no skills yet)**

Expected: `Built .well-known/agent-skills/index.json with 0 skills`

- [ ] **Step 4: Verify the JSON is valid**

Expected: Valid JSON with `$schema` and empty `skills` array.

### Task 6: GitHub Pages workflow

**Files:**
- Create: `.github/workflows/pages.yml`

Deploys the `.well-known` directory and skill files via GitHub Pages so agents can discover skills at `https://klaushougesen.github.io/skills/.well-known/agent-skills/index.json`.

- [ ] **Step 1: Create GitHub Pages workflow**

```yaml
name: Deploy Skills to GitHub Pages

on:
  push:
    branches: [main]
    paths:
      - 'skills/**'
      - '.well-known/**'
      - 'scripts/**'
  workflow_dispatch:

permissions:
  contents: read
  pages: write
  id-token: write

concurrency:
  group: "pages"
  cancel-in-progress: false

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Rebuild discovery index
        run: bash scripts/build-index.sh

      - name: Prepare Pages artifact
        run: |
          mkdir -p _site
          cp -r .well-known _site/.well-known
          cp -r skills _site/skills

      - name: Upload artifact
        uses: actions/upload-pages-artifact@v3
        with:
          path: _site

  deploy:
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    runs-on: ubuntu-latest
    needs: build
    steps:
      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v4
```

- [ ] **Step 2: Commit**

